# MA Motors

A complete professional car dealership website built with Next.js, TypeScript, Tailwind CSS and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Included

- Dark luxury automotive design with gold accents
- Home, stock, vehicle details, sell your car, finance, about, contact and reviews pages
- Responsive sticky navigation and mobile menu
- Vehicle stock database structure in `lib/vehicles.ts`
- Search and filters for make, model, price, year and mileage
- Finance calculators
- Enquiry and valuation forms ready to connect to a backend, CRM or email provider
- WhatsApp floating button and vehicle-specific WhatsApp CTA
- Google Maps embed
- SEO metadata per page
